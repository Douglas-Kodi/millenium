# Millenium social network developed in laravel and vue

## How to use

### Install npm

```sh
npm install
```

### Download the files from GitHub

- Create a folder
- Open cmd
```sh
git clone --repositorie-link
```

### How to start BD

- Create BD name "millenium" and collation "utf8mb4_unicode_ci"
- Open cmd
```sh
php artisan migrate
```
- Start BD

### Starter Millenium

- Open cmd
```sh
php artisan serve
npm run dev
```
