# Millenium social network developed in laravel and vue

## How to use

### Install npm

```sh
npm install
```

### Download the files from GitHub

- Create a folder
- Open cmd
```sh
git clone --repositorie-link
```

### Starter BD

- Open file ".env" 
```sh
DB_DATABASE=millenium
```
- Open cmd
```sh
php artisan migrate
```

### Starter Millenium

- Open cmd
```sh
php artisan serve
npm run dev
```
